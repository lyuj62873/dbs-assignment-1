Design Process Journal

Fun Free Web Games Hub

A 25-iteration design exploration of a browser-based free web games directory. Starting from radical visual experiments, converging toward a refined, production-ready UI. All pages are built with pure HTML and internal CSS — no JavaScript, no frameworks.

Total Versions 25
Tech Stack HTML + CSS only

Combining, inverting, and referencing. Styles are crossed, game aesthetics are borrowed, and directions pushed in unexpected ways — fusion, horror, 1-bit logbooks, dungeon roguelikes, and post-apocalyptic Pip-Boy terminals.
